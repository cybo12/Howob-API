# MMO_API

### Introduction
API Rest and communications server for Howob game. Include a Discord server bot.

### Requirements
* flask 
* flask-RestFul
* redis  
* ansicolors 
* requests 
* pika
* python-twitter
* pyspeedtest




### Docker Guide

Howob and Discord can build by launching build.sh in Docker and requirements folders.
